**Imports of Modules used**


2. ```SpaCy Medium English Model``` For Text Processing
```
python -m spacy download en_core_web_md

```

* Root Folder

  * JSON Input Folder

**File path to be entered must be ```RELATIVE``` to ```search.py```**
